**PR Title:** Add XDR Snip — lightweight screenshot tool for Windows

**Entry to add:**
```markdown
* [XDR Snip](https://github.com/db-cynerg-ia/xdr-snip) — Lightweight screenshot tool for Windows 11 with frozen screen overlay, JPEG output, and system tray. Single exe, no dependencies. [![MIT](https://img.shields.io/badge/license-MIT-blue.svg)](https://opensource.org/licenses/MIT)
```

**PR Body:**
XDR Snip is a lightweight screenshot tool for Windows 11 built in pure Rust. It uses Win32/GDI APIs via the `windows` crate for screen capture with a frozen overlay, encodes to JPEG via the `image` crate, and provides clipboard integration via `arboard`. Single executable (~11MB), no installer needed.

```markdown
- [XDR Snip](https://github.com/db-cynerg-ia/xdr-snip) - Lightweight screenshot tool with frozen screen overlay and JPEG output. Single exe, no installer. ![Open-Source Software](https://opensource.org/licenses/MIT)
```

**PR Body:**
XDR Snip is a lightweight, open-source screenshot tool for Windows 11. Press Print Screen to freeze the screen, drag to select a region, and get a JPEG (200-800KB) instead of an uncompressed bitmap (6MB+). Single portable exe, MIT licensed.
